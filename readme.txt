
Python application that implements K-Means clustering for
customer segmentation. The program reads data from an external 
JSON file, preprocesses it, applies K-Means clustering, 
 visualizes the results. It also saves the clustered data 
 to a CSV file.

Clustering enables better customer understanding and segmentation for strategic planning.

Output
Console:
Silhouette Score for evaluating cluster quality.
Confirmation that clustered data has been saved.
CSV File:
A file named clustered_customers.csv containing original customer data with an additional Cluster column.
Visualization:
A scatter plot showing customer segments in reduced dimensions using PCA.
